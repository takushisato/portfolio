# React + TypeScript + Vite で作成したポートフォリオ

## 公開 URL

```
# github.io
https://takushisato.github.io/portfolio/
```

## よく使うコマンド

```
# 環境インストール
npm install

# ローカルサーバー起動
npm run dev

# ビルド
npm run build

# github.io にデプロイ
npm run deploy
```
